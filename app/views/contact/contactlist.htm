Contact List Notes

A few notes on the structure of this page and what views, methods and such are used at each step.

Intiial View: On startup - the table rows are dynamically added to the empty table via an ajax call to the getContacts() method.

Create Contact: When you click the add button, it opens the modal dialog with the contact form. An action with the value of "add" tells the form we are adding a new contact. Upon submit, the data is written to the database via and ajax call to the addContact() method.

Edit Contact: The edit button fires off a js function to load the contents of the row's cells into the main modal window input table, then it opens the modal window. On submit - that data is sent via agax to the updateContact() method.

Delete Contact:The delete button simply calls the js script to delete the record via the deleteContact() method.

Animations: Some of the code in this example is there to fade rows in and out. Obviously this isn't critical to the function but I wanted to add it to make it more of a real-world example.
